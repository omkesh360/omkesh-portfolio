Kaun Banega Crorepati

Indian quiz game with lifelines and authentic KBC experience

Game Features

  • 15 progressive difficulty questions
  • Prize ladder from ₹1,000 to ₹1 Crore
  • Three lifelines: 50-50, Phone a Friend, Ask the Audience
  • Authentic KBC sound effects
  • Timer for each question
  • Safe checkpoints at ₹10,000 and ₹3,20,000
  • Quit option to secure winnings

Technical Implementation

  • Dynamic question loading system
  • Interactive lifeline mechanics
  • Progress tracking and scoring
  • Audio integration for effects
  • Responsive UI design
  • Local storage for game state
  • Animation and transition effects

Lifelines

  • 50-50: Removes two wrong options
  • Phone a Friend: Get help from a virtual friend
  • Ask the Audience: See audience poll results
  • Each lifeline can be used only once
  • Strategic usage increases winning chances

Prize Structure

  • ₹1,000 - ₹80,000 (Questions 1-8)
  • ₹1,60,000 - ₹6,40,000 (Questions 9-12)
  • ₹12,50,000 - ₹1,00,00,000 (Questions 13-15)
  • Safe points at ₹10,000 and ₹3,20,000
  • Quit anytime to secure current winnings
