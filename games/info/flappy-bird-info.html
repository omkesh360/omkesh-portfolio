Flappy Bharat Game

Indian-themed flying game with tricolor kite and cultural elements

Game Features

  • Indian tricolor kite as the main character
  • Cultural background themes
  • Collectible items with Indian motifs
  • Progressive difficulty levels
  • Smooth animations and controls
  • High score tracking
  • Responsive design for all devices

Technical Details

  • Built with HTML5 Canvas
  • Pure JavaScript game engine
  • CSS3 animations and transitions
  • Collision detection system
  • Local storage for high scores
  • Mobile-responsive controls
  • Optimized performance

How to Play

  • Click or tap to make the kite fly
  • Avoid hitting the obstacles
  • Collect special items for bonus points
  • Try to beat your high score
  • Share your achievements

Design Elements

  • Indian flag color scheme
  • Traditional patterns and motifs
  • Cultural background elements
  • Smooth gradient effects
  • Modern UI with Indian touch
